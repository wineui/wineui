Wine UI

A lightweight and modular front-end framework
for developing fast and powerful web interfaces.

Let's taste the wine.

Features

Wine UI gives you a comprehensive collection of HTML, CSS, and JS components
which is simple to use, easy to customize and extendable.

Sass

Wine UI is developed in Sass to write well-structured, extendable code which is easy to maintain.

Components

A collection of small, responsive components using consistent and conflict-free naming conventions.

Customizer

Wine UI very basic style can be extended with themes and is easy to customize to create your own look.

Responsive

With the mobile-first approach Wine UI provides a consistent experience from phones and tablets to desktops.

Open Source

Wine UI is an open source framework to make the web look good.
Create your own styles and discover the unlimited use of Wine UI.

MIT License

Wine UI is open source and MIT licensed. It is absolutely free of charge and you can use, copy, merge, publish and distribute the framework without any limitations.
